## 📁 Projets Phares

### 🌤️ [Weather Data Pipeline](https://github.com/votre-pseudo/weather-data-pipeline)
Un pipeline ETL automatisé conçu pour extraire, nettoyer et stocker des données météorologiques en temps réel.
- **Stack :** Python, Apache Airflow, PostgreSQL, Docker, API REST.
- **Point clé :** Orchestration entièrement conteneurisée pour une gestion fluide des flux de données.

### 🌐 [2MCSA - Infrastructure Multi-Site](https://github.com/votre-pseudo/2mcsa-infrastructure)
Conception et déploiement d'une infrastructure Windows Server sécurisée pour une entreprise multi-site.
- **Stack :** Windows Server, Active Directory, GPO, DFS, Tunnels VPN.
- **Point clé :** Automatisation des partages réseau et de la gestion des droits utilisateurs.

### 🎮 [Power Quest](https://github.com/votre-pseudo/power-quest)
Un jeu de stratégie en temps réel (RTS) mettant l'accent sur une architecture UI/UX propre et un système de localisation.
- **Stack :** Godot 4 (GDScript), Serveur Nakama (backend multijoueur).
- **Point clé :** Intégration d'un système de traduction dynamique basé sur des fichiers CSV et animations d'interface fluides.
